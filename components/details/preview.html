FAQ Example
What is a details element?

The HTML details element creates a disclosure widget in which information is visible only when the widget is toggled into an "open" state.

When should I use it?

Use details elements for FAQs, collapsible sections, or any content that should be hidden by default.

Is it accessible?

Yes! The native HTML details element is fully accessible and works with keyboard navigation and screen readers.
